# Assembler Tetris

<a alt="DosBox"> <img src="https://img.shields.io/badge/DosBox-yellow" /></a>

To run this game use DOS-BOX. Use all the commands under DOS-BOX into the project directory.

- ##### Compilation
```
TASM.EXE pog.asm
```
- ##### Linking
```
TLINK.EXE POG.OBJ
```
- ##### Running
```
POG.EXE
```
